# Instructions
To compile the project:
    1. Unzip the n26-nestorTheInvestorFolder
    2. Enter in and make: "npm i" in the linux terminal
    3. When it finishes without errors then put: "npm run dev"
    4. Copy the local link and put it in the browser.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
